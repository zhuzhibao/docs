# 企业资质

## 简要描述

* 添加企业资质

## 请求URL

* `{{domain}}/api/business/qualification-company`

## 请求方式

* POST 

## 参数

| 参数名 | 必选 | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| qualification\_id | 是 | int- | 资质ID |
| expired\_at | 是 | int- | 到期时间 |
| url | 是 | int- | 上传地址 |

## 返回参数说明

| 参数名 | 类型 | 说明 |
| :--- | :--- | :--- |
| code | int | code |
| data | int | 返回数据 |
| msg | string | 信息 |
